### 🏆 QApilot — Agentic AI 기반 통합 테스트 자동화 시스템
> **SKALA AI 서비스 개발 팀 프로젝트 최우수상(1위)** · **기술 리드** — Orchestrator·Agent 설계·백엔드·테스트 대상 시스템(SUT) 전체 담당 · SKALA 3기 AI 1반 3조(6인) · 발주: SKT NOVA개발3팀·ICT서비스팀

코드베이스와 요구사항 문서(PRD·정책·약관)를 읽어 **E2E 테스트 시나리오(TS·TC·TV)를 자동 생성**하고, Playwright로 **UI–API–DB 3-Tier를 교차 검증**하는 Agentic AI 통합 테스트 자동화 시스템입니다. 장애 발생 시 원인 후보(Top-N)와 수정 코드·담당자를 제안하고, 요구사항–테스트 추적 매트릭스(RTM)·HITL 승인까지 전 과정을 자동화합니다. (통상 3~4개월 걸리는 통합 테스트 대상)
- 💡 골든셋 **224 TC** 실측 — **요구사항 커버리지 100% · 결함 판정 정확도 96.6% · 테스트 코드 정확도 89.2%**
- ⚙️ 시나리오 생성을 **TS·TC·TV Agent로 분리**해 컨텍스트 폭주·할루시네이션 완화 → **LLM 호출 비용 약 93%↓**($0.21→$0.0144, usage_metadata 실측)

- 🧱 테스트 대상 시스템 **Mini-BSS**(통신 BSS MSA: FastAPI×2 + Vue 3, 의도적 결함 7종 골든셋)까지 직접 설계·구현
